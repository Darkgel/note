初始化

vue init webpack my-project
cd my-project
npm install

npm run lint -- --fix

配置使用scss
npm install -D node-sass sass-loader

添加normalize.css
npm install --save normalize.css

        import 'element-ui/lib/theme-chalk/index.css'
